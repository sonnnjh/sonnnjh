중고거래 오이마켓

회원제 중고거래 사이트
중고물품을 게시하고, 검색하고, 수정하고 삭제할 수 있다. 
로그인 후 모든 기능을 사용한다.
회원은 아이디와 이름과 위치를 저장한다.

프로그램 시작 후 첫 메뉴 1. 가입하기  2. 로그인 후 쇼핑하기

업그레이드1>
게시물 검색은 비회원도 가능하다.
게시, 수정, 삭제는 회원만 가능하다.

프로그램 시작 후 첫 메뉴   1.가입하기  2. 쇼핑하기
쇼핑하기를 선택하면 1. 검색  2. 등록  3. 수정   4. 삭제  5. 전체보기

업그레이드2>
삭제는 게시자만 가능하다.

프로그램 시작 후 첫 메뉴   1.가입하기  2. 쇼핑하기
쇼핑하기를 선택하면 1. 검색  2. 등록  3. 수정   4. 삭제  5. 전체보기

업그레이드3>
각 회원은 콕하기 기능이 있다.
콕하기 기능은 관심 게시물을 관리하는 기능이다. 

프로그램 시작 후 첫 메뉴   1.가입하기  2. 쇼핑하기
쇼핑하기를 선택하면 1. 검색  2. 등록  3. 수정   4. 삭제  5. 전체보기
검색하기 기능을 선택하면 검색 물품의 정보가 출력되고, 키보드로 콕하기 게시물을 등록한다.
